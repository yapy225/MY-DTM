    Comment s’inscrire dans un annuaire ?

  • S’inscrire dans l’annuaire
    Vous devrez commencer par inscrire votre site internet dans l’annuaire dans la catégorie de votre activité.

  • Ajouter votre site sur l’annuaire
    L’inscription de votre site internet faite, ajouter votre site web dans l’annuaire. Identifiez votre site internet par une description unique, originale en y ajoutant des mots clés pertinents et ciblés afin d’obtenir des liens retour. Afin d’éviter le duplicate content et évité d’être sanctionner par Google. La rédaction de votre description devrait être faite avant votre inscription dans l’annuaire.
  • Renseignez les champs par les informations de votre entreprise
    Renseignez les champs par les informations de votre entreprise permettront de renseigner les gens sur vos cordonnées, adresse, numéro de téléphone, d’ouverture et de fermeture.
  • Ajouter visuels
    Ajouter des photos ou des vidéos à la présentation de votre entreprise afin que vos potentiels clients puissent facilement vous identifier.
  • Lien vers votre site
    Afin d’accroitre du trafic vers votre site internet, vous devrez créer un lien qui fasse le pont en l’annuaire et votre site internet.
  • Vérification
    Votre site internet sera soumis à une vérification une fois votre inscription terminé.
